# 💰 Controle de Gastos (MVP)

Aplicação web desenvolvida como MVP (Produto Mínimo Viável) para controle de finanças pessoais.
Permite registrar, visualizar, editar e remover gastos de forma simples e dinâmica.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 📌 Funcionalidades

✅ Adicionar gastos com:

* Descrição
* Valor
* Categoria

✅ Listagem dinâmica dos gastos

✅ Cálculo automático do total gasto

✅ Remoção de itens da lista

✅ Edição de gastos

✅ Destaque visual para valores acima de R$100,00

---

## 🖥️ Interface

A aplicação é uma **Single Page Application (SPA)**, ou seja, todas as interações acontecem em uma única página sem recarregamento.
